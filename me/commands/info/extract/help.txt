 SYNOPSIS
  Extract archives

 USAGE
  extract [options] file[s]
 
 DESCRIPTION
  Extract archive[s], suported types (xz, gz, bz2, zip, rar)


 OPTIONS

  --no-verbose  Don't be verbose
  --to-dir=     Extract into `dir'
  --strip       Extract archive without the leading directory
  --help        Show help
  --info        Show information
