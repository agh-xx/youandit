 SYNOPSIS
  Help for the distribution

 USAGE
  help options

 DESCRIPTION
  Show help for using the application

 OPTIONS
 
  --intrinsic           pcre pattern to look for intrinsic function
  --functionkeys        Show F[keys] functions
  --buildintrinsiccache Build cache to look for intrinsic functions
  --pcresyntax          Show pcresyntax
  --myself              Show general help about the application
  --help                Show this message
  --info                Show information
